# Hospital #
 

## 软件介绍 ##
  医院管理系统分为患者、医生和管理员三个角色。患者可预约挂号、拿药缴费；医生可以接诊、开药和发送信息给管理员。管理员可管理医生、科室、药品，并查看所有所有订单。另外，患者有手机App软件可预约挂号和缴费。

## 采用技术 ##
- 医院管理系统前端使用Html+css + Javascrip +jQuery + mui + 正则表达式 + Ajax
- 医院管理系统后台使用JavaWeb
- 数据库使用Oracle
- 调用了百度地图的API，调用了发送短信验证码的API
